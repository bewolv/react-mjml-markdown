Combine mjml and markdown to generate responsive emails.

Replace mjml layout in `/pages/generatehtml.js` with your layout keeping `mj-style` and `mj-raw` tags.

Development mode:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

Production mode:

```bash
npm install
npm run build
npm run start
# or
yarn
yarn build
yarn start
```

Press `Ctrl/Cmd + S` to reload preview and html
